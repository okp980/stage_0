### SETUP INSTRUCTION

1. Clone repo from git
2. cd into `stage_0` folder
3. Create virtual environment with `pythom3 -m venv venv` for mac(check for windows if using windows)
4. Acitviate virtual environment with `source venv/bin/activate` for mac(check for windows if using windows)
5. Run `pip install -r requirements.txt` to install dependencies
6. Run `fastapi dev main.py`
